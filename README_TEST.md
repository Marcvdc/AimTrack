# Testbestand

Dit bestand is aangemaakt als onderdeel van een geautomatiseerde test om de GitHub Actions-agent te valideren.

## Doel

Dit bestand dient ter verificatie dat de Claude Code-agent correct:

- Bestanden kan aanmaken in de repository
- Wijzigingen kan pushen naar een nieuwe branch
- Een pull request kan openen met een correcte beschrijving

## Status

Succesvol gegenereerd door de Claude Code GitHub-automatiseringsagent.
